## Description
Interactive web application allowing users to watch TV channels online.  
The project was developed with AI assistance to generate code and design components, demonstrating modern web development skills.

## Features
- Selectable TV channels with live streaming
- Responsive web interface
- Clean and readable code structure
- AI-assisted development for faster prototyping

## Technologies
- HTML5
- CSS3
- JavaScript (ES6)
- AI tools used for coding support
